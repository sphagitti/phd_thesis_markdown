\chapter{The ARIMA model}\label{sec:intro}}

The ARIMA model consists of three different models combined into one and
can be used to make predictions.

\hypertarget{ar-part}{%
\section{AR part}\label{ar-part}}

Values from previous time stamps are used to predict the value for the
next time stamp. The parameter \textbf{p} determines how many past
values are used. Thus, the formula for the AR model can look like this:

outcome of the prediction. If not desired, it can be set to 0 and be
left out.
